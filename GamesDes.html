Hero's Trail


Heros Trail is a part of the projects that was made as part of my IT course within the Doncaster UTC. This was made to show and make purpose of the skills learnt within the unit and for this the assignment was to research and make a prototype to a game that was unique and our own idea. This came with doccumenting the research and then the plan my project went through several ideas before landing on a 2D dungeon crawler.

I began with the template for this game on the GameMaker program page and then I adapted it using its sprites and assets, I hid and added in several other coins hidden in and around the level and since it was a prototype it was only one level but this can be built upon and changed upon user testing. This was done in the process and to change was some UI changes that were mocked up but they didnt fit the timescale to implement them. The changes were controls on the UI along with more, harder enemies.
